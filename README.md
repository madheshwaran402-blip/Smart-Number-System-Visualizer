# Smart Number System Visualizer & Analyzer

An enhanced Java-based Number System Converter developed as part of the SkillBuildEra Java Development Internship.

## Features

- Decimal to Binary Conversion
- Decimal to Octal Conversion
- Decimal to Hexadecimal Conversion
- Binary Visualization
- Prime Number Detection
- Odd/Even Analysis
- Power of Two Check
- Bit Usage Analysis
- Conversion History Tracking
- Export Report to File

## Technologies

- Java
- IntelliJ IDEA
- File Handling
- Collections Framework
- OOP Concepts

## Sample Output

Displays a detailed Number Analysis Report including:

- Decimal
- Binary
- Octal
- Hexadecimal
- Prime Status
- Odd/Even Status
- Power of Two Status
- Binary Visualization
